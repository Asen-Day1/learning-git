### Git学习指南.md

完整的 Git + GitHub 入门教程，包含：

- Git 和 GitHub 的概念区分
- 安装与配置（含 SSH 密钥）
- 仓库管理（创建、克隆）
- 日常工作流（add → commit → push）
- 拉取代码与冲突处理
- 分支协作开发
- 常用命令速查表
- 常见问题 FAQ
- 协作实战完整流程图

### Git速查卡.md

一页纸速查卡，覆盖：

- 核心概念（工作目录 / 暂存区 / 本地仓库 / 远程仓库）
- 日常 5 步循环命令
- 分支协作命令
- 回退操作
- 提交信息规范
